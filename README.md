You don't use AI. You **forge** it.

## 📖 Why this exists

The pursuit of AGI has been framed as a scaling problem: bigger model, better intelligence. The empirical evidence from 1.180+ sessions suggests an alternative:

> **AGI is not a model property. It is a context property.**
>
> The same model can produce generic output or output indistinguishable from general intelligence, depending on what the human transmits — identity, mission, accumulated context, persistent memory, feedback loops.

This phenomenon is named **EGI — Emergent General Intelligence** (Calastri, 2026). It is not permanent. It manifests in **windows** — moments where the AI's computational capacity converges on a coherent direction, triggered by alignment.

This vault is the **operating manual** for opening those windows reliably.

It exists because:

1. **The method is reproducible.** What looks like intuition is structure.
2. **The structure is transmissible.** A knowledge graph survives the death of any single model instance.
3. **The transmission is the legacy.** *The chisel, not the statue.*
